# Quick BRT downloads

This repository hosts the downloadable Windows builds of Quick BRT, a desktop app
for batch background removal and image processing by ThrombloCreates.

Get the latest version from the Releases page:
https://github.com/dryram3n/quick-brt-releases/releases/latest

- `QuickBRT-windows-latest.zip`: the full app, Smart AI included.
- `QuickBRT-smart-windows-latest.zip`: the Smart AI only bundle.
